| "What $89 buys you vs what $0 buys you" | Side-by-side comparison | Visual contrast |

## Content Production SOP

```
1. Script      — AI-drafted via n8n workflow or manual (30-60 second target)
2. Record      — Phone, 15-60s, natural lighting, casual tone, vertical format
3. Edit        — CapCut (add captions, trending audio for TikTok/IG)
4. Post TikTok — Native upload (algorithm favors native > cross-posted)
5. Re-edit IG  — Adjust caption style, 20+ relevant hashtags
6. X thread    — Extract key points into thread format (3-7 tweets)
7. Schedule    — Queue via Postiz (social.filefree.tax)
```

## Engagement Rules

1. **Reply to every comment** in the first 60 minutes (algorithm signal)
2. **Pin the best comment** on each TikTok (drives conversation)
3. **Reply with video** to interesting questions (TikTok feature)
4. **Follow back** early followers who are in the Gen Z tax niche
5. **Never delete negative comments** unless abusive — respond transparently
6. **Stitch/Duet** with creators talking about tax pain points
