FileFree + LaunchFree — logo direction round 2

Research dossier for fresh exploration. No final direction is selected. References are links only (no hotlinked images) so this page works offline and avoids broken embeds. Palette hexes are locked from docs/brand/PROMPTS.md (no separate PALETTES.md file).

Context: round 1 used droplet-family geometry (still reflected in brand.mdc and shipped SVGs). Founder asked to move away from droplet / water / fluid cues for a “could be completely different” pass.

Founder ratification (decision support)

No network submission — use shortlist checkboxes on cards, then capture decisions here.

Parent P1–P5 prompts in PROMPTS.md stay locked. This exploration does not change the paperclip system.

Rejection signal (contrast): The prior round’s droplet + center-dot pattern reads as a shared “water bead” family; the founder called that exploration unsuitable. This round prioritizes archival, receipt abstraction, typographic, gate/threshold, and velocity-without-rocket languages instead — while staying inside locked product hexes.

FileFree — 10 candidate directions (Indigo + Lime)

Product: bookkeeping / receipt-to-tax; audience: solopreneurs. Avoid droplet, water, fluid. Primary #4F46E5 · accent #84CC16.

Tabbed Archive

Object

A row of folder or binder tabs reduced to a single clean mark — one tab lifts or aligns forward (lime) so “your pile becomes navigable.”

Reference set (open URLs — 4–8 sources)

Browser/file UI “tabs” context — Wikipedia: Tab (interface)
Structured-doc positioning — Wikimedia Commons: Notion logo (SVG file page)
Museum of design objects / organization culture — Cooper Hewitt Smithsonian Design Museum (homepage)
Visual mood for “filing / order” (search) — Unsplash: filing cabinet results
Palette

Why it could work

  • Instant “files / sections” read for a tax + receipt product.
  • Scales to favicon as stacked rectangles; animation: tab nudge or slide.
  • Clearly not droplet territory.

Why it might fail

  • Risk of looking like a generic browser/IDE icon without strong proportion rules.
  • Can collide with “tabs” in UI chrome if mark is too literal.

Nano Banana Pro prompt sketch

1024×1024 transparent. Flat vector. Three slim folder tabs in a row; left tab slightly forward. Indigo #4F46E5 strokes; one continuous lime #84CC16 segment on the leading tab edge only. Rounded caps. ~60% frame. No text, no gradients, no 3D, no water/droplet shapes. Style: Notion, Linear, Lucide. Single accent zone (sub-product rule).
